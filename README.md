# Baby Sleep Tracker MVP (Next.js)

Минимальное web-приложение для трекинга сна и роста ребенка.

## Что умеет (MVP)

- **Профиль ребенка**: добавить ребенка, переключать активного.
- **Сон**:
  - кнопка **старт/стоп** (дневной/ночной сон)
  - если запустить новый сон при уже идущем — текущий сон **автоматически закрывается** текущим временем.
  - **редактирование/удаление** записей сна на экране «День».
- **Рост/вес**:
  - добавление замеров
  - простой график «вес по возрасту» (упрощенные линии P3/P50/P97 для MVP)
- **Статистика**: средний сон/день (7 дней), среднее окно бодрствования, самый длинный сон (7 дней)
- **Экспорт в Excel**: один файл `.xlsx` с листами `Children`, `Sleep`, `Growth`.

## Технологии

- Next.js (App Router) + TypeScript
- IndexedDB через **Dexie** (локальное хранение)
- Экспорт `.xlsx` через **xlsx**

## Запуск

```bash
npm i
npm run dev
```

Откройте: http://localhost:3000

## Данные

Хранятся **локально в браузере** (IndexedDB). Авторизация в MVP не используется.

## Замечания про «ВОЗ график»

В MVP используются **упрощенные референсные точки** (пример). Для точного совпадения со стандартами ВОЗ подключите полные таблицы WHO Weight-for-age (LMS/percentiles) и пересчитайте кривые.
